HTML Tags

Text Tags

Paragraph Tag - <p>

A paragraph tag is used to describe a paragraph.

Heading Tags - <h1> ~ <h6>

The heading tags denote heading, where lower numbers are parent headings and higher numbers are subheadings.

Preformatted - <pre>

Preformatted text allows us to intrepret text
                        based on format utilized in the defining
                                    documents.

Code - <code>

Denotes a section of code.

Strong, Emphasis - <strong>, <em>

Strong and emphasis provide tools for providing emphasis to text written on a webpage. This tag both changes the underlying semantic meaning of the affected text, and is also levaraged by user-agent sheets to bold/italicize text.

List Tags

Ordered List - <ol>

An ordered list is a list in which we are communicating that the order items are presented in is meaningful and that it should be preserved. Generally, user-agent stylesheets (browser default styles) will present order lists with numbers.

  1. First item
  2. Second item
  3. Third item

Unordered List - <ul>

A list in which the elements of the list are ordered arbitrarily and the order has no significant meaning. Consequently, systems altering the order is permissible.

  • An item
  • Another item
  • Yet another item

Description List <dl>

Description list is different in that each item has parts, some item and its description. The item is defined with <dt> and its description is defined with <dd>

Apple
A sweet fruit that grows on a tree.
Oranges
In no way comparable to apples.

List Item - <li>

List item tag is used as a child tag of an order list or Unordered list, representing an individual item in that list. Lists of type ol/ul should not have any direct children that are not list items.

Multi-media

Image Tag <img>

Images are tags that can be used to load an image and display it as part of the webpage. It's important to understand that generally images are not a part of the website themselves, they are loaded as a resource. As such, we must always include the location of the resource that we intend to load. As such, we need a way to provide additional information to the tag.

Additional information can be provided to tags using what are called attributes. Attributes are names or key-value pairs that can be utilized by the tag. For instance, images leverage a src attribute to derive the location of the resource it needs to load.

Video Tag <video>

Video tag is used to provide HTML support for loading and viewing videos.

Audio Tag <audio>

Multimedia tag used to play audio files.

(Anchor tag) Hyper-reference <a>

Anchor tags provide a link to another location on a network (or the internet). Anchor tags form hyperlinks which are imperative to the purpose and meaning of HyperText. HyperText is the idea of immediate and unending information (in the form of text).

Forms

Forms are used to gather data from the user of the page, using a variety of input types. These allow the user to interact with the page by setting values and then submitting these values to a server. Note: At the current time we have no server, so these values won't be submitted to any particular place. Later, we will have our own servers that we can submit requests to.

Form Tag - <form>

Form tag is a containing tag that holds a group of input fields and represents a group of related units of data that should be submitted together.

Input Tag - <input>

The input tag represents an input element. The type of input is defined using a type attribute. For instance, we might use the 'text' type to submit text fields, or the radio button to submit an option.
